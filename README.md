# Personal Blog Platform

## Project Overview

This is a React.js internship project built using Vite and React Router.

The project contains:
- Home Page
- Article Page
- Author Page
- Admin Dashboard
- Reusable Design System
- Dark Mode Toggle

---

## Technologies Used

- React.js
- Vite
- React Router DOM
- Axios
- React Markdown

---

## Features

- Dynamic Routing
- Reusable UI Components
- CSS Design Tokens
- Dark Mode with localStorage Persistence
